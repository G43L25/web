WEB

La web es un sistema de información en Internet que permite acceder, compartir y vincular contenidos mediante hipertexto, visualizados a través de navegadores.

HTML

HTML, acrónimo en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado utilizado en la creación de páginas web.

CSS

CSS es un lenguaje informático especializado en definir y cohesionar la presentación de un documento escrito en un lenguaje de marcadocomo HTML o XML.

JAVASCRIPT

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web, como animaciones, botones, formularios y actualizaciones en tiempo real.
