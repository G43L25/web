Introduccion

En informática, la World Wide Web (La Web) o red informática mundial​ es un sistema que funciona a través de internet, por el cual se pueden transmitir diversos tipos de datos a través del Protocolo de Transferencia de Hipertextos o HTTP, que son los enlaces de la página web.

Historia

Fue creada entre 1989 y 1990 por Tim Berners-Lee, con el apoyo de Robert Cailliau, mientras trabajaban en el CERN. Inspirada en conceptos como el hipertexto de Ted Nelson y el sistema Memex de Vannevar Bush, Berners-Lee logró combinar el hipertexto con Internet, creando el primer navegador, servidor y páginas web. La primera página web, lanzada en 1991, explicaba el funcionamiento de la Web y su propósito. En 1993, el CERN hizo pública esta tecnología, marcando un hito en la accesibilidad de la información. Posteriormente, navegadores como Mosaic, desarrollado en 1993, popularizaron la Web gracias a su interfaz gráfica. Este avance permitió que la WWW se convirtiera rápidamente en el protocolo de Internet más utilizado.

Estructura de una página web

El frontend se refiere a la parte visible de un sitio web o aplicación con la que los usuarios pueden interactuar directamente. Es la cara del sitio, ubicada en el lado del cliente. Por otro lado, el Backend se encarga de la conexión con la base de datos y el servidor utilizado por el sitio web.

Tecnologias principales

Las tecnologías clave para el desarrollo web incluyen HTML para la estructura, CSS para el diseño y JavaScript para la interactividad. Frameworks como React y Node.js facilitan la creación de interfaces y funcionalidades avanzadas. Las bases de datos como MySQL almacenan y gestionan la información, mientras que las APIs permiten integrar servicios externos. Estas herramientas juntas crean sitios web dinámicos y funcionales, adaptados a las necesidades de los usuarios.
