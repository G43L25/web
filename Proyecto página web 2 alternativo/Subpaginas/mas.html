Recursos

GITHUB:GitHub es una plataforma de control de versiones y colaboración que permite a los desarrolladores almacenar y gestionar su código, así como colaborar con otros en proyectos de software. Visita HitHub

Overflow Stack: Overflow es una comunidad de programadores donde puedes hacer preguntas, compartir conocimientos y encontrar soluciones a problemas comunes de programación Visita Overflow

Codecademy: Codecademy ofrece cursos interactivos para aprender HTML, CSS, JavaScript y otros lenguajes de programación. Es ideal para principiantes y aquellos que desean mejorar sus habilidades. Visita CodeAcademy

W3Schools: es una referencia en línea para aprender HTML, CSS, JavaScript y otros lenguajes web. Ofrece ejemplos de código. Visita W3Schools

YouTube: YouTube tiene muchos tutoriales y cursos gratuitos sobre desarrollo web. Puedes encontrar canales dedicados a enseñar HTML, CSS, JavaScript y otros lenguajes. Visita YouTube

Herramientas

Virtual Studio Code:Es un editor de código gratis que soporta varios lenguajes de programación.

Sublime Text:Es un editor de texto sofisticado para código, marcado y prosa.

Atom:Editor de texto código como vs code, creado por GitHub.

React: Es una biblioteca de JavaScript que es para construir interfaces de usuario.

Angular:Una plataforma de desarrollo para construir aplicaciones web móviles y de escritorio.

Trello: Una herramienta de gestión de proyectos para organizar tareas.

Jira: Es una herramienta para la gestión de proyectos y seguimiento de errores.
