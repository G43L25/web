Introducción

CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets).Es un lenguaje que maneja el diseño y presentación de las páginas web.Funciona junto con el lenguaje HTML que se encarga del contenido básico de los sitios.

Historia

Las hojas de estilos aparecieron poco después que el lenguaje de etiquetas SGML, alrededor del año 1970. Desde la creación de SGML, se observó la necesidad de definir un mecanismo que permitiera aplicar de forma consistente diferentes estilos a los documentos electrónicos. El organismo W3C (World Wide Web Consortium), encargado de crear todos los estándares relacionados con la web, propuso la creación de un lenguaje de hojas de estilos específico para el lenguaje HTML y se presentaron nueve propuestas. Las dos propuestas que se tuvieron en cuenta fueron la CHSS (Cascading HTML Style Sheets) y la SSP (Stream-based Style Sheet Proposal). La propuesta CHSS fue realizada por Håkon Wium Lie y SSP fue propuesto por Bert Bos. Entre finales de 1994 y 1995 Lie y Bos se unieron para definir un nuevo lenguaje que tomaba lo mejor de cada propuesta y lo llamaron CSS (Cascading Style Sheets).

Selectores Básicos

  • Selector de elementos: Se aplica a todas las etiquetas de un tipo específico, por ejemplo, p { color: blue; } para todos los párrafos.
  • Selector de clases: Aplica estilos a elementos con una clase específica, usando . seguido del nombre de la clase, como .button { background-color: red; }.
  • Selector de ID: Aplica estilos a un único elemento con un ID específico, usando # seguido del nombre del ID, por ejemplo, #header { font-size: 24px; }.
  • Selector universal: Selecciona todos los elementos de la página con *, por ejemplo, * { margin: 0; }.
  • Selector de atributos: Aplica estilos a elementos que tienen un atributo específico, por ejemplo, input[type="text"] { border: 1px solid black; }.

Codigo simple

/* Estilo del cuerpo */
body {
       font-family: sans-serif;
 }
/* Estilo del encabezado */
h1 {
       color: blue;
 }
/* Estilo de los párrafos */
p {
       color: gray;
 }
